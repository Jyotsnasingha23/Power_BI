# Power_BI
Orders ReportSales Review Dashboard

An interactive dashboard designed to analyze sales performance, profit trends, and product distribution across regions, categories, and time.

---

 Key Features

-  Sales, Profit & Quantity KPIs
-  Category & Sub-category analysis
-  Regional filtering (Central, East, South, West)
-  Monthly sales trends
-  Interactive and dynamic visuals

---

 Key Insights

- Technology category generates the highest sales
- Phones and Chairs are top-performing sub-categories
- Sales show noticeable growth in later months (seasonal trend)
- High sales volume but relatively lower profit margin
- Some sub-categories like Supplies and Paper underperform

---

 Dataset

- Source: Superstore dataset
- Includes:
   - Orders data
   - Sales & Profit
   - Region
   - Category & Sub-category

---

Objective

The goal of this project is to:

- Identify top-performing products and categories
- Analyze regional sales distribution
- Track monthly sales trends
- Discover opportunities to improve profitability

- 💰 Advanced profit margin breakdown
- 🌐 Deployment to web dashboard
